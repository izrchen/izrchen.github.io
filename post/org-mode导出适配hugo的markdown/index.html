ox-hugo#

安装#

  1. 打开包管理： M-x package-list-packages
  2. 搜索包： C-s ox-hugo
  3. 安装： install

配置#

在 org-mode export dispatch 中加入快捷键：

;; ox-hugo
(with-eval-after-load 'ox
  (require 'ox-hugo))

使用#

  1. 输入hugo所需要markdown头部
  2. 导出markdown到hugo博客目录
